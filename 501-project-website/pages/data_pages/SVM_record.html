SVM Classification using Record Data¶

Introduction¶

In this section, I will introduce the process of applying the Support Vector Machine (SVM) to record data classification, the dataset I use for SVM classfication, as well as the the classfication results.

The record data I will be using for SVM classification is the country fertility rate dataset (FertilityRate_Clean.csv) that was cleaned and used in the previous sections, which can be found here. The fertility rate dataset contains 10 attributes, which are Country Name, Country Code, year, Fertility Rate, Region, Income Group, GDP per Capita in USD, Human Development Index, Teritary School Enrollment Percentage, and labels. The labels of this dataset are binary that contain two classes: "Below Replacement" and "Above Replacement".

A snapshot of the dataset will be shown in the class distribution section below.

def generate_label_data(class_labels, weights, N=10000):
    # e.g. class_labels=[0,1] weights=[0.2, 0.8] (should sum to one)
    
    y = random.choices(class_labels, weights = weights, k = N)
    print("-----GENERATING DATA-----")
    print("unique entries:",Counter(y).keys())
    print("count of labels:",Counter(y).values()) # counts the elements' frequency
    print("probability of labels:",np.fromiter(Counter(y).values(), dtype=float)/len(y)) # counts the elements' frequency 
    return y
    
    
 ## RANDOM CLASSIFIER
def random_classifier(y_data): 
    ypred=[]
    max_label=np.max(y_data); #print(max_label) 
    for i in range(0,len(y_data)):
        ypred.append(int(np.floor((max_label+1)*np.random.uniform(0,1))))
    print("-----RANDOM CLASSIFIER-----")
    print("count of prediction:",Counter(ypred).values()) # counts the elements' frequency
    print("probability of prediction:",np.fromiter(Counter(ypred).values(), dtype=float)/len(y_data)) # counts the elements' frequency
    print("accuracy",accuracy_score(y_data, ypred))
    print("percision, recall, fscore,",precision_recall_fscore_support(y_data, ypred))

Using random classifier on the fertility data based on the weights calculated above to get accuracy, precision, and recall values. In this part, I used 0 to represent the label "positive", 1 as "negative", and 2 as "neutral".

Conclusion¶

Based on the classification reports above, we can see that the predictive performance of the Support Vector Machine (SVM) classifier on the testing data was relatively satisfying. Among all the kernels used for the SVM, RBF kernels and linear kernels have the best predictions. Overall speaking, SVM models fit the fertility rate dataset ideally.
